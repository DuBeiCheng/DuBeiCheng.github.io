伪类：条件

}

p:nth-of-type(1){

5.4、父级边框塌陷问题

- clean

/*
clear：right；右侧不允许有浮动元素
clear：left； 左侧不允许有浮动元素
clear：both； 两侧不允许有浮动元素
clear：none； 
 */
.layer04{
    border: 1px #666 dashed;
    font-size: 12px;
    line-height: 23px;
    display: inline-block;
    float: right;
    clear: left;
}

解决方案
- 1、增加父级元素的高度

#father{
    border: 1px #000 solid;
    height: 800px;
}

- 2、增加一个空的div（class=“clear”）标签，清除浮动
